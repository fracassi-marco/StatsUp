## Come sostituire le immagini

Se desideri sostituire le immagini con altre personalizzate:

1. Prepara immagini in formato WebP (consigliato) o PNG/JPG
2. Dimensione consigliata: 800x600px o superiore
3. Rinomina le immagini secondo lo schema `bg_[tipo_attività].webp`
4. Sostituisci i file nella cartella `app/src/main/res/drawable/`
5. Le immagini saranno automaticamente utilizzate dall'app

## Ottimizzazione

Le immagini WebP sono già ottimizzate per dimensione e qualità. Se utilizzi immagini PNG o JPG, considera di convertirle in WebP per migliori prestazioni.
