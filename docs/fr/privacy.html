Politique de Confidentialité

Dernière mise à jour : 20 mars 2026 · S'applique à : StatsUp pour Android

En bref : StatsUp stocke toutes vos données uniquement sur votre appareil. Nous ne collectons, ne transmettons ni ne vendons aucune information personnelle. Nous ne gérons pas de serveurs. Nous n'utilisons pas de SDK d'analyse ou de publicité. Les seuls services externes contactés par StatsUp sont Intervals.icu (pour importer vos activités) et Google Maps (pour afficher les parcours).

1. Qui nous sommes

StatsUp est une application Android développée et maintenue par Marco Fracassi (fracassi.marco@gmail.com). L'application est open source et disponible sur GitHub.

2. Données auxquelles nous accédons

StatsUp demande un accès en lecture seule à votre compte Intervals.icu afin d'importer vos activités d'entraînement. L'application lit notamment :

  • Données d'activité : distance, durée, dénivelé, allure, vitesse, fréquence cardiaque, type, date
  • Coordonnées GPS/parcours associées aux activités
  • Votre nom de profil Intervals.icu et photo de profil (affichés localement dans l'application)

StatsUp ne demande jamais l'accès en écriture à Intervals.icu. Elle ne peut créer, modifier ou supprimer aucune donnée sur votre compte Intervals.icu.

3. Où vos données sont stockées

Toutes les données importées depuis Intervals.icu sont stockées exclusivement dans une base de données SQLite locale sur votre appareil (via la bibliothèque Room). Ces données ne quittent jamais votre appareil sauf si vous les exportez explicitement via la fonctionnalité de sauvegarde intégrée.

StatsUp n'a pas de serveurs backend. Il n'y a pas de système de compte, pas de synchronisation cloud et pas de base de données distante.

4. Partage des données

Nous ne partageons pas vos données avec des tiers, avec les exceptions techniques suivantes :

  • API Intervals.icu — lorsque vous connectez votre compte ou déclenchez une synchronisation, l'application communique avec les serveurs de Intervals.icu pour récupérer vos données d'activité. Ceci est régi par la Politique de Confidentialité de Intervals.icu.
  • Google Maps — lorsque vous visualisez une carte de parcours, les coordonnées GPS sont envoyées à Google Maps pour afficher les tuiles de carte. Ceci est régi par la Politique de Confidentialité de Google.

5. Analyses et publicité

StatsUp ne contient aucun SDK d'analyse, aucun service de rapport de plantage, aucun SDK publicitaire ni aucun type de suivi. Nous n'avons aucune visibilité sur la façon dont vous utilisez l'application.

6. Autorisations

StatsUp demande les autorisations Android suivantes :

  • INTERNET — nécessaire pour communiquer avec l'API de Intervals.icu et Google Maps.
  • ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION — nécessaires pour afficher vos itinéraires GPS sur la carte.
  • FOREGROUND_SERVICE / FOREGROUND_SERVICE_DATA_SYNC — nécessaires pour exécuter le service d'importation en arrière-plan pendant l'utilisation de l'application.

7. Conservation et suppression des données

Vos données existent uniquement sur votre appareil. Vous pouvez supprimer toutes les données à tout moment en utilisant l'option "Resynchroniser depuis Intervals.icu" (qui remplace toutes les données locales) ou en désinstallant l'application, ce qui supprime complètement la base de données locale.

Pour révoquer l'accès de StatsUp à votre compte Intervals.icu, visitez Intervals.icu → Paramètres → Mes applications et déconnectez StatsUp.

8. Confidentialité des enfants

StatsUp n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment d'informations auprès des enfants. Si vous pensez qu'un enfant a utilisé cette application et avez des préoccupations, veuillez nous contacter.

9. Modifications de cette politique

Si nous apportons des modifications importantes à cette politique, nous mettrons à jour la date en haut de cette page et, le cas échéant, noterons les modifications dans les notes de version de l'application.
